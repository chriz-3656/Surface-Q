Core Capabilities

Security Capabilities.

Everything you need to passively analyze, scan, and monitor attack surfaces.

Passive Scanning

Analyzes publicly broadcasted headers, security policies, and third-party script links without invasive network probes.

Gemini & Groq Dual-AI

Evaluates security posture profiles using Google Gemini and Groq AI (Llama 3.3) for automated remediation guides.

Interactive Threat Chat

Direct conversational chat with SurfaceQ AI in the dashboard to request vulnerability mitigations and CSP configs.

Simple & Expert Audits

Toggles dashboard metrics between clear, color-coded risk ratings (Simple) and technical headers/JSON lists (Expert).

Dynamic Site Diagnostics

Automated classification heuristics tracing dynamic Server-side SSR routing cookies vs. static CDN configurations.

Privacy First Architecture

Stores all logs and scanned domain footprints locally. Zero remote telemetry is ingested during checking.

Privacy by Design

Complete transparency about what we analyze and what we never touch.

What We Analyze

  • HTTP headers (CSP, HSTS, security flags)
  • SSL certificates (expiration, encryption keys)
  • DOM structure & visible tags
  • Script sources & external references
  • Meta tags and SEO fields
  • Technology fingerprints and versions

What We Never Collect

  • Passwords or login credentials
  • Personal identity variables (PII)
  • Cookies, auth sessions, or local storage
  • Form inputs and database queries
  • Browser extensions telemetry traffic
  • Authentication tokens and access keys
